Conhecimentos

*Passe o cursor do mouse no card para ler* HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar e estruturar conteúdo na web. O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação e o layout de documentos HTML. JavaScript é uma linguagem de programação essencial para tornar páginas web interativas. Responsável pela dinâmica e funcionalidades, permite interações do usuário e manipulação do conteúdo da página em tempo real. React é uma biblioteca Javascript com foco em criar interfaces de usuário de forma componentizada Typescript é um superset do Javascript que inclui recursos que não estão presentes na linguagem de forma nativa, além de torná-la estática Node.js é um ambiente de execução JavaScript do lado do servidor, permitindo criar aplicativos web escaláveis e rápidos. MySQL é um sistema popular de banco de dados relacional, usado em muitos aplicativos web e empresariais pela sua confiabilidade e desempenho.
